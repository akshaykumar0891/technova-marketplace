## 3. Production Deployment Guide

Since you mentioned you are publishing this project online, follow these steps to deploy it:

### Step 1: Deploy the Database (Cloud MySQL)
1. Register on a database hosting provider (e.g., [Aiven](https://aiven.io/), [Clever Cloud](https://www.clever-cloud.com/), or [ScaleGrid](https://scalegrid.io/)).
2. Create a new **MySQL** database instance.
3. Note the database connection variables (Host, Port, User, Password, DB Name).

### Step 2: Deploy the Backend API (Render / Railway)
1. Push your project folder to a GitHub repository.
2. Sign in to [Render](https://render.com/) and create a new **Web Service** linked to your backend repository.
3. Configure build/start parameters:
   - **Root Directory**: `backend`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
4. Set up the following **Environment Variables** in Render's dashboard settings:
   - `DB_TYPE` = `mysql`
   - `DB_HOST` = *(Your Cloud MySQL Host)*
   - `DB_USER` = *(Your Cloud MySQL Username)*
   - `DB_PASSWORD` = *(Your Cloud MySQL Password)*
   - `DB_NAME` = *(Your Cloud MySQL DB Name)*
   - `DB_PORT` = `3306` (or database port)
   - `JWT_SECRET` = *(Generate a long secret string)*
   - `PORT` = `10000` (Render allocates this automatically)
5. Save. Your backend will build and start, creating the tables in the MySQL database. Copy the backend API url (e.g., `https://technova-api.onrender.com`).

### Step 3: Deploy the Frontend SPA (Vercel / Netlify)
1. Sign in to [Vercel](https://vercel.com/) and create a new project linked to your GitHub repository.
2. Configure build parameters:
   - **Root Directory**: `frontend`
   - **Framework Preset**: `Vite`
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
3. Set the following **Environment Variable** in Vercel's settings:
   - `VITE_API_URL` = `https://technova-api.onrender.com/api` *(Your live Render backend URL, appended with /api)*
4. Click **Deploy**. Vercel will build your static files and publish them to a live domain name (e.g., `https://technova-marketplace.vercel.app`).


### Admin User Setup Guide
Since default demo accounts have been deleted for production deployment, follow these steps to add your first administrator:
1. Launch the site, navigate to `/register`, and create a regular account.
2. Log into your database console and execute:
   `UPDATE users SET role = 'admin' WHERE email = 'your_registered_email@example.com';`
3. Log in again on the site to access the Admin Console.
